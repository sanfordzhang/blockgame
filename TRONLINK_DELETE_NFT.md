## TronLink删除NFT方法

### 方法1: 通过设置删除
1. 打开TronLink
2. 点击右上角 **设置图标** ⚙️
3. 找到 **管理NFT** 或 **NFT管理**
4. 找到PANFT合约，点击删除

### 方法2: 直接刷新（推荐）
不需要删除，直接刷新即可：

1. 切换网络：主网 → Nile测试网
2. 返回NFT收藏品
3. 下拉刷新

### 方法3: 清除缓存
1. TronLink设置
2. 高级设置
3. 清除缓存
4. 重启TronLink

### 验证Cards显示
刷新后，点击NFT #10查看属性，应该显示：
- Cards: 10h 9d 8c 7s 6h
