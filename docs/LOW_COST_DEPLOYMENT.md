# 💰 低成本主网部署方案（≤500元）

## 📊 实际成本分析

### TRX 价格参考
- 当前价格：约 0.15 USD = 1.1 RMB
- 500 元预算 ≈ **450 TRX**

---

## 🎯 优化方案

### 方案 1: 能量租赁（推荐）✅

**部署阶段：**
- 租赁 500,000 能量（1天）：约 30 TRX
- 部署合约消耗：约 300,000 能量
- 剩余能量可用于初始化操作
- **成本：30 TRX ≈ 33 元**

**运营阶段：**
- 租赁 1,000,000 能量/天：约 50 TRX/天
- 可支持约 20-30 次合约调用/天
- 月成本：1,500 TRX ≈ 1,650 元
- **日均成本：50 TRX ≈ 55 元**

**初期投入（7天运营）：**
- 部署：30 TRX
- 运营（7天）：350 TRX
- 预留：50 TRX
- **总计：430 TRX ≈ 473 元** ✅

---

### 方案 2: 质押获取能量

**部署阶段：**
- 质押 30,000 TRX 获得能量
- 部署完成后赎回（3天解冻期）
- **实际消耗：0 TRX（仅锁定3天）**

**运营阶段：**
- 质押 50,000 TRX 获得持续能量
- 可支持日常运营
- **实际消耗：0 TRX（长期锁定）**

**问题：需要大量 TRX 锁定，不适合初期**

## 🔧 具体实施步骤

### 步骤 1: 准备资金

```bash
# 购买 TRX
币安购买：500 TRX（含缓冲）
成本：约 550 元
```

### 步骤 2: 租赁能量

**推荐平台：**
1. **TronEnergy.market** - 最便宜
2. **JustLend** - 官方平台
3. **TronNRG** - 稳定可靠

**租赁步骤：**
```
1. 访问 TronEnergy.market
2. 选择租赁 500,000 能量
3. 租期：1 天
4. 支付：约 30 TRX
5. 能量会自动到账
```

### 步骤 3: 部署合约

```bash
# 确保有足够能量
node check-energy.js

# 部署合约
npm run deploy:mainnet

# 预计消耗：300,000 能量
```

### 步骤 4: 日常运营

**每天租赁能量：**
- 根据实际使用量调整
- 低峰期：500,000 能量（25 TRX）
- 高峰期：2,000,000 能量（100 TRX）

## 💡 进一步优化建议

### 1. 合约优化
- 移除不必要的事件
- 减少存储变量
- 优化数据结构

### 2. 批量操作
- 多个玩家一起结算
- 减少合约调用次数

### 3. 链下处理
- 游戏逻辑在服务器处理
- 只在必要时调用合约

### 4. 使用代理模式
- 一次部署，多次升级
- 节省重复部署成本
